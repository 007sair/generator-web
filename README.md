# 这是一个简易的脚手架工具

> 为什么会有这个项目？

因为我已经受够了不停的新建文件夹、到处拉取文件、修改gulpfile、package.json文件 +_+

## 包含功能

- sass转换css
- 自动添加css3前缀
- css格式化、强力压缩
- browser-sync实时刷新
- js压缩
- H5使用750or640设计稿自由切换

## 使用方法

**安装：**

```
npm install
```

ok，这样你就创建好了一个自己的新项目

**执行：**

```
gulp
```

然后开始coding吧~~~
